**command** *-options* *argument*

Опции модифицируют поведение комманды.
Аргумент - объект, с которым работает комманда.

-a  -  короткий формат опции
--all  -  длинный формат
-lt  -  комбинирование опций (длинный формат отображения,  сортировка по времени)
