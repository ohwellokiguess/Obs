## Overload for ==
```c
auto operator==(const VectorInt& lhs, const VectorInt& rhs) {
	return lhs.GetLength() == rhs.GetLength();
}
```

## Overload for comparison operations by <=>
```c
auto operator<=>(const VectorInt& lhs, const VectorInt& rhs) {
	return lhs.GetLength() <=> rhs.GetLength();
}
```
